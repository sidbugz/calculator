# Calculator

A command line calculator built in Python.

## What it does
- Adds, subtracts, multiplies, and divides two numbers
- Handles invalid input gracefully using error handling
- Built as part of my 12-month developer learning journey

## How to run it

Make sure you have Python installed, then run:
```bash
python calculator.py
```

## What I learned building this
- Python basics: variables, data types, input/output
- Decision making with if/elif/else
- Error handling with try/except
- Debugging by reading error messages
